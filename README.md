# 青少年團契活動庫

這是一個單檔 HTML 版的青少年團契活動資料庫，整理 40 個兼具趣味性與信仰扣回的活動。

## 功能

- 搜尋活動、材料、主題
- 分類篩選
- 動態程度篩選
- 難度篩選
- 隨機推薦
- 收藏活動
- 點開活動查看：
  - 玩法流程
  - 需要預備的材料
  - 事前預備
  - 信仰扣回
  - 收束問題
  - 注意事項

## 使用方式

直接開啟 `index.html` 即可使用。  
若部署到 GitHub Pages，網站網址通常會是：

```text
https://你的GitHub帳號.github.io/你的repo名稱/
```

## 更新方式

之後若要新增活動，可以在 ChatGPT 中貼上新活動內容，再請 ChatGPT 更新 `index.html`。
